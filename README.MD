# IBM-GenAI-Engineering
Professional portfolio for the IBM Generative AI Engineering program. Features end-to-end RAG pipelines, LLM fine-tuning, and AI Agents using LangChain, Watsonx, and Python.

This repo is where I’m tracking my progress. 
I’m spending 5 hours a day learning how to move past simple chat prompts and actually build tools that use AI to solve real problems.
Instead of just "talking" to an AI, I'm learning the behinds of it:

"Giving AI "Memory": Using Vector Databases so the AI can remember and search through specific documents (RAG).
Connecting Tools: Using LangChain to let the AI actually do things, like search the web or run code.
Fine-Tuning: Teaching a model how to speak or act in a specific way for a specific job.
Real Apps: Moving my code out of a notebook and into a real, clickable app using Flask and Gradio.

My 4-Week Plan
Week 1: The Basics. Prompt engineering and how LLMs actually work. (NOT STARTED)
Week 2: Coding. Deep dive into Python and building the "skeleton" of AI apps.
Week 3: The Hard Stuff. RAG, Vector DBs, and advanced AI architecture.
Week 4: The Grand Finale. My Capstone project—building a complete AI tool from scratch.
